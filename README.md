## How to Use

1. Open `index.html` in your web browser
2. Select your country from the dropdown menu
3. Click "Check Today's Holidays" to see if today is a public holiday
4. View holiday details including name, type, and description

## Technologies Used

- HTML5
- CSS3 (with modern gradients and animations)
- Vanilla JavaScript (no dependencies)

## Local Development

Simply open the `index.html` file in any modern web browser. No build process or server required!

```bash
# Clone the repository
git clone https://github.com/VassilAtanasov/PublicHoliday.git

# Navigate to the directory
cd PublicHoliday

# Open in browser
# On macOS:
open index.html
# On Linux:
xdg-open index.html
# On Windows:
start index.html
```
